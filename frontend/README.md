# Expense Tracker Frontend

## Overview

Frontend for the Expense Tracker application using React, Tailwind CSS, Chart.js, and JWT-based auth.

## Structure

- `src/components` - reusable UI components
- `src/pages` - page-level views and dashboard screens
- `src/context` - app state and auth context
- `src/services` - API client and data services
- `src/utils` - helpers and formatting utilities
- `src/hooks` - custom React hooks
- `src/App.jsx` - app shell and routing
- `src/index.jsx` - React entry point

## Setup

1. Create `.env` in `frontend/`
2. Install dependencies: `npm install`
3. Run app: `npm start`
